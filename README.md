# 🚀 JN Marketing Management System

## 🏢 Company Overview
JN Marketing is a product distribution and sales management system designed to manage:

- 📦 Product Orders
- 👥 Team Hierarchy
- 📊 Sales Reports
- 🕒 Attendance System

This system is built as a web-based corporate management application.
